# Git Training Assignment

## What I Learned

During this Git and GitHub training, I learned the basic concepts of version control and how Git helps manage projects efficiently.

### Task 1: Repository Setup

* Created a local Git repository using `git init`.
* Created a GitHub repository and connected it to the local repository.
* Configured the remote repository and pushed the project to GitHub.

### Task 2: Initial Commit

* Created a simple website using HTML and CSS.
* Added project files to the staging area using `git add`.
* Created the first commit using `git commit`.
* Pushed the initial commit to the GitHub repository.

### Task 3: Branch Management

* Created separate branches for different features.
* Worked on the following branches:

  * `feat/contact`
  * `feat/about`
  * `feat/login`
  * `style`
* Added changes and committed them on their respective branches.
* Pushed all branches to GitHub.

### Task 4: Git Cherry-Pick

* Learned the purpose of the `git cherry-pick` command.
* Understood how to copy a specific commit from one branch to another without merging the entire branch.

**Syntax:**

```bash
git cherry-pick <commit-hash>
```
